ACCESS & LOGIN PROTOCOL

To evaluate the system's role-based dashboards and functionality, please use any identity from the provided CSV files.

How to identify login credentials:
Administrative Staff: Use any email found in staff.csv (e.g., m.adams@birminghamgp.nhs.uk).
Medical Clinicians (GP, Nurse, Specialist): Use any email found in clinicians.csv (e.g., d.thompson@birminghamgp.nhs.uk).
Patients: Use either the Email or the NHS Number found in patients.csv (e.g., sarah.brown@email.com or 4567890123).
